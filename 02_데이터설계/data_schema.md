# 데이터 설계

## 수집 데이터 (크롤링)
| 컬럼 | 타입 | 설명 | 분석 활용 |
|------|------|------|----------|
| 상품명 | str | 상품 이름 | 텍스트 분석 |
| 브랜드 | str | 브랜드명 | 세그먼트 기준 |
| 카테고리 | str | 상품 분류 | 필터링 |
| 가격 | int | 판매가 (원) | AOV proxy |
| 평점 | float | 평균 별점 | 만족도 |
| 리뷰수 | int | 전체 리뷰 수 | CAS 구성 ×4 |
| 좋아요수 | int | 찜/좋아요 수 | CAS 구성 ×2 |
| 조회수 | int | 1개월 조회수 | CAS 구성 ×0.01 |
| 구매수 | int | 1년 구매수 | LTV proxy |
| 인기나이 | str | 주 구매 연령대 | 타겟 특성 |
| 인기성별 | str | 주 구매 성별 | 타겟 특성 |
| 태그 | str | 상품 태그 | 텍스트 분석 |
| 리뷰1~10 | str | 리뷰 텍스트 | 감성 분석 |

## 파생 변수
| 변수명 | 산출 공식 | 의미 |
|--------|----------|------|
| CAS_score | (리뷰수×4 + 좋아요수×2 + 조회수×0.01) 정규화 0~100 | 커뮤니티 활동 점수 |
| popularity_score | (구매수×3 + 평점×1000) 정규화 0~100 | 구매 인기도 |
| segment | CAS × popularity 조합 | 유저 유형 분류 |
